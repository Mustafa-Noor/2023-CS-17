# Smart Event Manager

## Overview
**Smart Event Manager** is an Android application designed to help users register for events easily. The app allows users to view a home screen, fill out an event registration form, upload a profile image, and view a confirmation summary of their registration.

This project was developed as part of **BS Computer Science, 3rd Year, 6th Semester** coursework.

---

## Features

- **Splash Screen:** Displays app logo, name, and welcome text for 2–3 seconds.
- **Main Screen:** Shows a welcome message, description, and an event banner with a "Register for Event" button.
- **Event Registration Screen:**
  - Form fields for full name, phone number, email.
  - Spinner for selecting event type.
  - Date picker to choose the event date.
  - Radio buttons for gender selection.
  - Image upload option for profile or event image.
  - Checkbox for accepting terms and conditions.
  - Form validation with Toast messages.
  - Confirmation dialog before submission.
- **Confirmation Screen:** Displays submitted registration details along with the uploaded image and a success message.
